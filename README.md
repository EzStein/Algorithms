This repo contains java source code for various implementations of several algorithms. Algorithms include data structures:
Lists:
	Singly Linked List
	Doubly Linked List
	Array List

Trees:
	Binary Search Tree
	Heap
	Treap
	N-tree
	Splay Tree
	AVL Tree
	2-3 Tree

Other:
	Hash Table (with variations)

Sorting Algorithms:
	Insertion Sort
	Quick Sort
	Quick Select
	Merge Sort
	Selection Sort
	Radix Sort
